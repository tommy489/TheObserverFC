About

The Observer FC

An independent football media platform built on a simple idea: that the game deserves better than clickbait. Tactical depth, honest transfers analysis, and long-form storytelling — this is The Observer FC.

We cover European football from the Premier League to La Liga, Serie A, Bundesliga, and the Champions League. Every article is written with genuine passion for the sport and a commitment to going beyond the headlines.

Founded in 2026, The Observer FC is for readers who want context, history, and real insight — not just another match recap.

🤖 Claude API — AI Digest & AI Excerpt
Get your free key at console.anthropic.com → API Keys → Create Key. Stored only in your browser (localStorage).
🗄️ Supabase — Real Database ⚡ Required — everyone sees articles from here
⚠️ Pour que tous les visiteurs voient les articles, tu dois coller ton URL et ta clé anon directement dans le fichier index.html, dans les constantes PUBLIC_SB_URL et PUBLIC_SB_ANON (en haut du script). Les clés dans ce formulaire ne sont stockées que dans ton navigateur.
One-time setup: Create a free account at supabase.com → New project → SQL Editor → run the SQL below → paste your keys here.
SQL — run once in Supabase SQL Editor 
create table articles (
  id uuid default gen_random_uuid() primary key,
  title text not null,
  slug text unique not null,
  cat text default 'Analysis',
  excerpt text default '',
  body text default '',
  image text default '',
  time text default '',
  status text default 'published',
  scheduled_at text default '',
  date text default '',
  created_at timestamptz default now()
);
alter table articles enable row level security;
create policy "Public read" on articles for select using (status = 'published');
For Twitter/Discord article image previews, also add these to Netlify → Site settings → Environment variables:
SUPABASE_URL = [your project URL]
SUPABASE_ANON_KEY = [your anon key] Also deploy the netlify.toml and netlify/edge-functions/og-inject.js files in your project root.
